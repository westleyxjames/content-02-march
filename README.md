## 🔧 TEMPLATE FOR NEW ARTICLE PAGES

Copy `articles/morning-habits.html` and modify:

1. Update `<title>` and `<meta name="description">`
2. Update article header (title, author, date, category)
3. Update featured image URL
4. Replace all `<div class="content-card">` sections with article content
5. Keep the same header/footer structure
6. Ensure back link points to `../index.html`

## 🔧 TEMPLATE FOR LEGAL PAGES

Copy `about.html` and modify:

1. Update `<title>` and meta description
2. Replace content cards with legal content
3. Use `<h2>` for sections
4. Use `<p>`, `<ul>`, `<li>` for content
5. Wrap each section in `<div class="content-card">`
